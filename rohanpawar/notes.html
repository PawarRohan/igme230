Design challenges

I used a clear, clean design. I wanted to only display a few pieces of information and let the projects themselves be the only defining color in the website. That required strong typography, which is why I chose to have such bold, large project headlines.

Above the call

Instead of finding a bit of javascript online to figure out how to make a dropdown menu, I tried to achieve the same effect from scratch. First, I tried to use :focus or :active tags in CSS to trigger the changes I wanted. Then, I used javascript to get a working dropdown menu. It was a challenge, but gratifying in the end.
