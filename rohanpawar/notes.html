Design challenges

I used a clear, clean design. I wanted to only display a few pieces of information and let the projects themselves be the only defining color in the website. That required strong typography, which is why I chose to have such bold, large project headlines.

Above the call

A challenge I found in this project was finding a way of using the minimalist, brutalist design scheme I wanted while making the website feel cohesive and unified. Making the menu, the home page, and project pages seem like part of the same design while differentiating between information was a challenge. In addition, creating a dropdown menu myself which was integrated and triggered with javascript commands was a gratifying challenge, using javascript to trigger CSS changes.
